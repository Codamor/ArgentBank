## Context

**Argent Bank** is an application which allows customer to check their accounts.

## Installation

### Prerequisites

- Node v18.10.0.

### Setup

Clone the repository, then :

#### Install and run back end
```
cd back-end
npm run install
npm run populate-db
npm run server
```

#### Install and run front end

```
cd front-end
npm run install
npm run start
```
